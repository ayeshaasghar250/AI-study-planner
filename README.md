# 📚 AI Study Planner

A desktop application built using Java and Swing that helps students automatically generate smart study schedules based on exam dates, subject difficulty, and daily study hours.


## 🚀 Features

* User Login & Registration
* Add Subjects with Difficulty Levels
* Automatic Study Schedule Generator
* Task Completion Tracking
* Progress Dashboard



## 🧠 How It Works

The system uses a rule-based logic to:

* Prioritize subjects based on difficulty
* Allocate study hours intelligently
* Adjust schedule based on exam proximity



## 🛠 Technologies Used

* Java
* Swing (GUI)
* Object-Oriented Programming (OOP)



## 📚 OOP Concepts Applied

* Encapsulation
* Inheritance
* Polymorphism
* Modular Design
